VaccaPropTable
--------------

class class vacca.properties.VaccaPropTable(parent=None, designMode=False)

   It is a class that inherits from TaurusPropTable, DoubleClickable
   (fandango) and Dropable (fandango).

   Shows the Devices Properties, this widget is listening to
   shareDataManager if any Device has been selected to show its
   properties

   This class has the follow functionalities:
      * Is connected to shareDataManager to share information in the
        GUI.

      * Can be edit the property by doubleclick function.

   static getDefaultIcon()

      Returns:
         The Default Icon Path.

   static getPanelDescription(name='TangoDeviceProperties', model='')

      Parameters:
         * **name** -- Name for the Panel

         * **model** -- Model for the panel

      Returns:
   setModel(model)

      Set Model is the callback used in shareDataManager to manage
      device selections. :param model: :return:
